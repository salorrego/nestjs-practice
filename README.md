## Description

This repository was made for user to learn to add testing

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test
> This is working on the branch `test`

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
